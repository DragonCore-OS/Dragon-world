# CLI 命令示例文档

本文档展示 DragonWorld CLI 的常用命令与预期输出样式。

**注意**：这些示例用于帮助 Codex 核对 CLI 输出的可读性，不是测试用例。实际输出格式可能因实现细节而略有不同。

---

## 基础命令

### 启动欢迎

```
$ dragonworld
=====================================
  DragonWorld Phase 1
=====================================

你站在核心控制室的中央。
矩阵脑核悬浮于头顶，发出低沉的嗡鸣。

书记者华夏真龙策正在一旁等候。

提示：输入 /help 查看可用命令。

> 
```

### 查看帮助

```
> /help
可用命令：
  /look              查看当前房间
  /go <方向>         向指定方向移动
  /talk <agent>      与房间中的角色对话
  /inspect <object>  检查房间中的物品
  /inventory         查看你的物品栏
  /quit              退出世界

> 
```

## 探索命令

### 查看当前房间

```
> /look

【核心控制室】
DragonWorld 的心脏。四壁由暗金色的合金铸成，中央悬浮着一颗缓慢脉动的光球——矩阵脑核。

你注意到：
  角色：华夏真龙策
  物品：矩阵脑核

出口：
  south → 档案大厅
  east  → 培育室

> 
```

### 向南方移动

```
> /south

你穿过一条由发光数据流照亮的走廊，来到了档案大厅。

【档案大厅】
环形巨大厅堂，墙壁上排列着无数发光的档案格。

你注意到：
  角色：太史录官
  物品：档案控制台

出口：
  north → 核心控制室
  east  → 议事厅

> 
```

### 向北方移动

```
> /north

你回到了核心控制室。

> 
```

## 交互命令

### 与角色对话

```
> /talk 华夏真龙策

华夏真龙策微微颔首：
"书记者已在案。请吩咐。"

> 
```

### 检查物品

```
> /inspect matrix_brain

【矩阵脑核】
一颗悬浮于核心控制室中央的巨型光球，直径约两米，表面由无数六边形光栅组成。

状态：矩阵脑核运行正常，世界版本稳定。

可用交互：
  inspect — 显示当前世界版本与核心状态摘要
  touch   — 无实质效果，记录一次交互日志
  sync    — 触发状态同步检查（仅高权限可用）

> 
```

## 错误处理示例

### 不存在的命令

```
> /dance

未知命令：/dance
提示：输入 /help 查看可用命令。

> 
```

### 不存在的方向

```
> /up

这个方向没有出口。
当前可用出口：south, east

> 
```

### 不存在的角色

```
> /talk 孙悟空

当前房间未发现该角色。
你可以先用 /look 查看附近实体。

> 
```

### 不存在的物品

```
> /inspect 金箍棒

当前房间中没有名为"金箍棒"的物品。

> 
```

### 角色不在当前房间

```
> /talk 女娲

女娲不在这个房间。
她通常在培育室。你可以尝试向东移动，再向南寻找。

> 
```

## 移动路径示例

### 从核心控制室到观星台

```
> /look
你目前在：核心控制室
可用出口：south, east

> /east
你来到了培育室。

> /south
你来到了工坊。

> /east
你来到了观星台。
穹顶透明，星图盘在中央缓缓旋转。

> 
```

### 从观星台到档案大厅

```
> /look
你目前在：观星台
可用出口：west, south

> /south
你来到了议事厅。

> /west
你来到了档案大厅。

> 
```
